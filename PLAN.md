# Nova2 重写开发计划（参考 issue #1）

## 1. 需求与边界
- 目标：在 Godot 4.6 环境下重新实现可运行的对话游戏框架，保留与 Nova2 相同的游戏叙事能力。
- MVP 目标：支持章节选择、对话播放、指令驱动的对象展示（show/hide/move/tint）、分支选择、跳转与章节结束。

## 2. 当前进度（本分支）
- 已完成：
  - `scene/game.tscn` 重建为 4.6 可直接加载场景。
  - 新增 `scripts/GameRoot.gd` 解析 `resources/scenarios/*.txt`，建立节点和事件流。
  - 支持 `label/is_start/is_unlocked_start/is_end/jump_to/branch` 等关键脚本指令。
  - 支持 `show/hide/move/tint/print` 简单动作。
  - 实现 `启动 -> 章节选择 -> 对话/分支 -> 结尾 -> 重开/退出` 流程。
- 待补：
  - 更完整的动作系统（动画链、摄像机位移、UI 框变化等）。
  - 动态语言与本地化。
  - 存档与恢复（checkpoint）。
  - 复杂的资源绑定（角色/场景节点绑定）。
  - 错误提示与日志系统。

## 3. 下一步开发顺序
1. **动作系统完善**：实现 3D/2D 显示对象统一驱动，补齐 `move/tint/show/hide` 的边界行为。
2. **剧情脚本解析增强**：支持更复杂参数、注释、空行策略、动作链组合。
3. **分支与跳转体验优化**：支持选择冻结、快速跳转、分支回退。
4. **章节/存档**：实现解锁规则、默认节点选择、读档/回档。
5. **UI/UX**：统一对话框/角色名样式、自动换行、快进逻辑。
6. **回归检查**：与原项目行为对齐，保证 `resources/scenarios/*.txt` 可继续执行。
